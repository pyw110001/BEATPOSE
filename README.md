## 🚀 核心特性 (Key Features)

### 1. ⚔️ 3D 光剑体感劈砍判定与多向音符
- **光剑体感指向**：通过摄像头（MediaPipe Hands 高精手部指骨追踪），将玩家双手镜像映射为 3D 空间中的红色（右手/屏幕左侧）与紫色（左手/屏幕右侧）双色发光光剑，支持高精度的指尖微动追踪。
- **3D 霓虹音符方块**：沿公路从消失点直飞而出的 3D 立方体音符带有随机的切击方向指示（上、下、左、右或任意点）。
- **3D 线段物理碰撞**：采用“线段（光剑）到圆心（音符）最短距离”检测模型，要求挥剑动作具有真实的物理速度（速度阈值 > 1.5），告别悬停作弊。
- **劈砍方向校验（Bad Cut）**：挥砍切入方向必须与方块上的箭头指向对齐（夹角小于 50°），若砍错方向会触发 `BAD DIRECTION!` 并重置连击。
- **切击分裂动画（Debris）**：击中时音符方块将一分为二飞散开来，并带有粒子火花爆破特效。

### 2. 🎵 本地音乐上传与自适应谱面
- **音乐时钟与网格对齐**：音符逻辑全面重构，所有判定时间均基于 **Music Clock** 与 **Beat Grid** 物理时钟计算，彻底对齐真实音乐的鼓点和重拍。
- **自定义 MP3/WAV 上传**：上传本地歌曲，系统通过 Web Audio API 异步解码并生成专属谱面。
- **Tap 对齐与偏移微微调**：大厅集成预览播放功能，支持在试听中通过 **Tap First Beat** 一键捕捉首个重拍，并支持 `±10ms` 精细微调，摆脱前奏静音错位。

### 3. 🔊 音量与音效全局配置
- **全局音效切换（SFX Toggle）**：顶栏控制区集成音效开关，支持一键静用/启用击中及 MISS 判定音效，配置实时生效并通过 LocalStorage 自动保存。

### 4. ⚡ 全局单例摄像头管理器 (Global Webcam Singleton)
- **0秒闪电载入**：摄像头硬件及 MediaPipe ML 模型采用全局生命周期托管，只在首次进入时初始化一次，后续切换关卡或进入标定“秒开”免去重复申请设备权限。
- **按需帧挂起（零额外 CPU 开销）**：返回大厅或模拟模式时，管理器会自动解除注册并全面挂起 ML 推理，在大厅状态下的 CPU/GPU 消耗完美归零。
- **手势指骨连线预览**：摄像头预览窗口内不再绘制笨重的全身骨骼，而是实时绘制 21 个手部关键点与 25 条指骨连线，带给您极客般的高科技感。

### 5. 🌐 中英文多语言一键切换 (i18n)
- 导航栏右上角内置一键切换按钮，支持整个大厅及游戏视口、Canvas 浮空字样、错误报告的全量中英文本地化，并自动持久化保存玩家选择。

## 🛠️ 本地运行与部署指南

### 本地运行 (Run Locally)

**环境要求：** Node.js 18+ 

1. **安装依赖**：
   ```bash
   npm install
   ```

2. **启动本地开发服务器**：
   ```bash
   npm run dev
   ```
   启动后可在浏览器中访问：`http://localhost:3000`

3. **静态类型检查 (Lint)**：
   ```bash
   npm run lint
   ```

## 📐 技术栈说明 (Tech Stack)
- **3D 渲染**：React Three Fiber (R3F) 9+, Three.js
- **2D 特效**：Canvas API (混合物理粒子系统)
- **框架**：React 19, TypeScript
- **构建工具**：Vite 6
- **样式**：CSS, TailwindCSS 4
- **体感视觉**：MediaPipe Hands, MediaPipe Camera Utils (CDN 加载)
- **动画效果**：Motion / AnimatePresence
- **图标**：Lucide React
- **音频处理**：Web Audio API (AudioContext)
